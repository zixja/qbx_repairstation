# qbx_repairstation

Modern vehicle repair stations for **Qbox** (QBCore fork) using Overextended resources.

Players can drive into designated zones and repair their vehicles using a clean, native-style menu powered by **ox_lib** and **ox_target**.

### Features
- Multiple configurable repair locations with blips
- ox_target interaction zones (box zones)
- ox_lib context menu with multiple repair options
- Different repair types: Full, Engine, Body, Tires
- Progress bar animation during repair
- Cash payment via qbx_core
- Configurable prices, durations, zones & debug mode
- Fully optimized & lightweight

### Dependencies
- ox_lib
- ox_target
- qbx_core
